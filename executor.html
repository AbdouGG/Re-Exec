    How to Use Re-Exec Ultra Pro

    1. Connect to the server:
      • The application will automatically attempt to connect to the server.
      • Check the connection status in the top-right corner of the editor.
    2. Write or load a script:
      • Use the code editor to write your Lua script.
      • Alternatively, load an existing script from the workspace panel.
    3. Execute your script:
      • Click the "Execute" button to run your current script.
      • Enable "Auto Execute" for automatic execution when receiving a Re-Exec signal.
    4. Manage your scripts:
      • Save your script using the "Save Script" button.
      • Organize scripts by categories.
      • Use the workspace panel to load, rename, or delete scripts.
    5. Customize your experience:
      • Access settings by clicking the "Settings" button.
      • Adjust theme, color scheme, font size, and other preferences.
    6. Import/Export scripts:
      • Use the "Export Scripts" button to save your scripts externally.
      • Use the "Import Scripts" button to load previously exported scripts.
